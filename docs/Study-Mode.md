# Study Mode

Study Mode is designed to help you learn the material with instant feedback.

## How It Works

1. Select your license class (Technician, General, or Extra)
2. Choose how many questions you want to study (configurable in Settings)
3. Answer each question
4. Get instant feedback on your answer

## Features

### Instant Feedback
- After selecting an answer, you'll immediately see if you were correct or incorrect
- The correct answer will be highlighted
- If you got it wrong, you must click the correct answer to continue

### Show Answer Option
In Settings, you can enable "Show Answer in Study" which:
- Shows the correct answer immediately when you select any answer
- Highlights both correct and incorrect answers
- Helps you learn faster

### Quick Study
In Settings, enable "Quick Study" to:
- Automatically advance to the next question after a correct answer
- Save time when you're practicing

## Navigation

- Use **Previous** to go back to a previous question
- Use **Next Question** to move forward
- Click **Finish** when you're done
- Use **Exit Exam** to leave the study session early

## Tips

- Take your time - there's no time limit in Study Mode
- Review your incorrect answers to understand the material better
- Use the "Show Answer" option if you're having difficulty
