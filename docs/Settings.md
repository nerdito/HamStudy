# Settings

The Settings page allows you to customize your study experience.

## Accessing Settings

Click the **Settings** button on the home page to access all settings.

## Settings Options

### Study Questions per License

Configure how many questions you want to study for each license class:

| License | Default | Minimum | Maximum |
|---------|---------|---------|---------|
| Technician | 25 | 5 | 411 |
| General | 25 | 5 | 427 |
| Extra | 25 | 5 | 602 |

Use the **-5** and **+5** buttons to adjust the number of questions.

### Quick Mode Settings

#### Quick Study
When enabled in Study mode:
- Correct answers automatically advance to the next question after 500ms
- Helps you move faster when you're confident

#### Quick Exam  
When enabled in Practice mode:
- Automatically advances to the next question immediately
- Simulates faster exam completion

### Display Settings

#### Font Size
Adjust the text size for better readability:

| Option | Size |
|--------|------|
| Small | 14px |
| Medium | 16px |
| Large | 18px |
| Extra Large | 20px |
| XX Large | 24px |

#### Show Answer in Study
When enabled in Study mode:
- Shows the correct answer immediately after selecting any answer
- Highlights both correct and incorrect answers
- Helps you learn faster

### Reset Options

#### Reset to Defaults
Resets all settings to their default values:
- Study Questions: 25 for each license
- Quick Study: Off
- Quick Exam: Off
- Font Size: Medium
- Show Answer: Off

#### Clear Progress
Clears all your exam history and progress data. This cannot be undone.

## Build Number

The build number is displayed at the bottom of the Settings page in the format:
```
yy.mm.dd.###
```

Example: `26.02.25.008` means February 25, 2026, build 008.

The sequential number resets daily.
